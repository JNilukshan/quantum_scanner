# Icon Assets

This directory contains icon files for the Quantum Scanner app.

## Recommended Icons

- **app_icon.png**: Main app icon in various sizes
- **qr_icon.svg**: QR code icon for UI elements
- **flash_icon.svg**: Flashlight toggle icons

## Usage

Add icon files to this directory and reference them in the Flutter code.

Example:
```dart
Image.asset('assets/icons/qr_icon.png')
```

## File Formats

- Use `.png` for raster icons
- Use `.svg` for vector icons
- Provide multiple resolutions for PNG icons (1x, 2x, 3x)
